This plugin can help you to share items placed on Ciaook on your WordPress site via [ciaook_share] shortcode or widget.

== Description ==
This plugin can help you to share items placed on Ciaook on your WordPress site via [ciaook_share] shortcode or widget. You can specify API Key as [ciaook_share api=xxx] or via Settings for shortcode. You can specify API Key in input field in widget administration or via Settings for widget.

== Installation ==
1. Unzip all files to the `/wp-content/plugins/ciaook_plugin` directory
2. Log into WordPress admin and activate the \'Ciaook Share Plugin\' plugin through the \'Plugins\' menu
3. Go to *Settings > Ciaook Share* in the left-hand menu to specify API Key from ciaook.com or set it via widget or set it via shortcode attribute [ciaook_share api=xxx]
4. Place shortcode [ciaook_share] to required posts or pages or place widget to required place/

== Frequently Asked Questions ==
= Where can I take API Key? =
You need to go to Ciaook.com to userpage and take API Key from API part - https://ciaook.com/index.php?page=userpage#user_api_key
